Modern To-Do App
A responsive, modern To-Do web application built with HTML, CSS (Tailwind), and JavaScript.

Features
Task Management:

Add new tasks

Mark tasks as complete (moves to Completed section)

Edit existing tasks

Delete tasks

Visual Organization:

Clear separation between Pending and Completed tasks

Count badges for each section

Date Tracking:

Shows when each task was added

Shows completion time for completed tasks

Persistence:

Tasks are saved to localStorage

Responsive Design:

Works on mobile and desktop devices

UI Enhancements:

Smooth animations

Font Awesome icons

Clean, modern design

Technologies Used
HTML5

CSS3 (with Tailwind CSS)

JavaScript (ES6)

Font Awesome (for icons)

Installation
No installation required! Simply open the index.html file in any modern web browser.

How to Use
Add a Task:

Type your task in the input field

Press Enter or click the "Add" button

Complete a Task:

Check the checkbox next to any pending task

It will automatically move to the Completed section

Edit a Task:

Click the edit (pencil) icon on any task

Modify the text in the popup modal

Click "Save" to update

Delete a Task:

Click the trash can icon on any task to remove it
