# 💻 Codex Techno Web Development & Designing Internship Tasks

Welcome to my GitHub repository for the Web Development and Designing Internship offered by **Codex Techno**. This repository contains all the tasks completed as part of the internship program, with each task explained in detail and implemented with proper documentation.

---

## 📋 Internship Overview

During this internship, participants were required to complete a series of web development tasks categorized into different levels. Each task helped build foundational to intermediate front-end development skills using **HTML**, **CSS**, and **JavaScript**.

Although the company required completion of only Level 1 tasks, I have completed **both Level 1 and Level 2 tasks** to enhance my learning and sharpen my skills beyond the basic requirements.

---

## ✅ Tasks Completed

### 🔹 Level 1 - Beginner Tasks

| Task No. | Task Name                 | Description                                                                                                             |
|----------|---------------------------|-------------------------------------------------------------------------------------------------------------------------|
| Task 1   | Landing Page              | Developed a static landing page using HTML and CSS with multiple sections like header, footer, and responsive layout. Focus on layout, spacing, and design consistency. |
| Task 2   | Portfolio                 | Created a personal portfolio website including introduction, resume section, project showcase, and contact information. |
| Task 3   | Temperature Converter     | Built an interactive web app to convert temperature between Celsius, Fahrenheit, and Kelvin with dropdowns, input fields, and output display. |

### 🔹 Level 2 - Intermediate Tasks

| Task No. | Task Name                 | Description                                                                                         |
|----------|---------------------------|---------------------------------------------------------------------------------------------------|
| Task 1   | Calculator                | Developed a basic calculator with addition, subtraction, multiplication, and division functionality. |
| Task 2   | Tribute Page              | Created a webpage honoring a notable personality with biography and images.                        |
| Task 3   | Basic To-Do Application   | Built a to-do list app with add, delete, and mark-as-complete features.                           |
| Task 4   | Login Authentication      | Implemented a login form with basic validation and user feedback.                                |

Each task contains its own **README.md** file with detailed instructions, features, and screenshots (where applicable).
