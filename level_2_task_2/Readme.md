# Dr. APJ Abdul Kalam Tribute Website

![Website Preview](https://images.unsplash.com/photo-1581093196277-1a43d36758c6?ixlib=rb-1.2.1&auto=format&fit=crop&w=1950&q=80)

A responsive tribute page dedicated to Dr. APJ Abdul Kalam, the "Missile Man of India" and beloved 11th President of India.

## Features

- **Modern, Responsive Design**: Works on all device sizes
- **Interactive Elements**: Hover effects, smooth scrolling, and animations
- **Comprehensive Sections**:
  - Biography and background
  - Key achievements
  - Life timeline
  - Inspirational quotes
  - Lasting legacy
- **Performance Optimized**: Fast loading with minimal dependencies
- **Accessibility Focused**: Semantic HTML and proper contrast ratios

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- Google Fonts (Poppins and Playfair Display)
- Pure CSS animations
- Responsive design with Flexbox and Grid

## Installation

No installation required! This is a static HTML/CSS website that can be:

1. Viewed directly in any modern web browser
2. Hosted on any static site hosting service (Netlify, Vercel, GitHub Pages, etc.)

To run locally:
```bash
git clone https://github.com/yourusername/kalam-tribute.git
cd kalam-tribute
# Open index.html in your browser
Usage
Simply navigate through the different sections using the menu or by scrolling. The website is designed to be intuitive and easy to navigate.

Sections Overview
Hero Section: Introductory banner with call-to-action

About: Biography and background information

Achievements: Key accomplishments in card format

Timeline: Chronological life events

Quotes: Inspirational sayings

Legacy: Impact and statistics about Dr. Kalam's influence

Customization
To customize the website:

Edit the HTML content in index.html

Modify colors by changing the CSS variables in the :root selector

Update images by replacing the URLs in the img tags

Add more quotes or timeline items by duplicating existing elements
