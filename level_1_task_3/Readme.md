Temperature Converter
Overview
Ye ek simple web-based Temperature Converter application hai jo user ko temperature ko ek unit se doosri unit mein convert karne ki suvidha deta hai. Ye Celsius (°C), Fahrenheit (°F), aur Kelvin (K) units ko support karta hai.

Features
Temperature input as a number
Conversion between Celsius, Fahrenheit, and Kelvin
Responsive and user-friendly UI with modern design
Error handling for invalid inputs
How to Use
Input field mein apna temperature value enter karein.
"From" dropdown se temperature ki current unit select karein (Celsius, Fahrenheit, ya Kelvin).
"To" dropdown se desired unit select karein jisme aap conversion karna chahte hain.
Convert button par click karein.
Niche result display hoga jisme converted temperature dikhai dega.
Technologies Used
HTML5 for structure
CSS3 for styling (with gradient background and modern UI)
JavaScript for temperature conversion logic
Code Explanation
JavaScript function convertTemp() pehle input value aur units ko read karta hai.
Agar input valid number nahi hai to error message dikhata hai.
Pehle input temperature ko Celsius mein convert karta hai (agar zarurat ho).
Fir Celsius se target unit mein convert karke result display karta hai.
How to Run Locally
Is project folder ko apne local machine par download karein.
index.html file ko kisi bhi modern web browser mein open karein (Chrome, Firefox, Edge, Safari).
Temperature converter use karna shuru karein.
